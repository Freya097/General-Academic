## 🚀 Getting Started

### Prerequisites

| Language | Tool Required |
|----------|--------------|
| C | GCC Compiler (`gcc`) |
| C++ | G++ Compiler (`g++`) |
| Python | Python 3.x interpreter |

### Running Programs

**C Programs:**
```bash
gcc filename.c -o output
./output
```

**C++ Programs:**
```bash
g++ filename.cpp -o output
./output
```

**Python Programs:**
```bash
python3 filename.py
```
